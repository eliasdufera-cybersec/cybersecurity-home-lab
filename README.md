# cybersecurity-home-lab – SOC Analysis & Security Testing

This repository documents experiments, reports, and progress from my cybersecurity home lab environment.

The lab is designed to simulate real-world security scenarios including attack simulations, threat detection, and Security Operations Center (SOC) analysis.

## Objectives

- Practice SOC investigation and incident response
- Perform attack simulations in a controlled lab environment
- Develop detection rules and security monitoring techniques
- Analyze logs and alerts using security tools

## Lab Focus Areas

- SOC Alert Investigation
- Threat Hunting
- Detection Engineering
- Attack Simulation
- Log Analysis
- Security Tool Testing

## Lab Architecture

The environment includes virtual machines simulating attacker systems, victim machines, and monitoring infrastructure such as SIEM and logging systems.
